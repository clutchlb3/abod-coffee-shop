# Coffee Shop Website

Phase 2 project for CSCI390 Web Programming

## About
This is a simple coffee shop website built with React and Tailwind CSS. 
The site has 4 pages: Home, About, Menu, and Contact. 
Made it responsive so it works on mobile and desktop.

## How to Run
```bash
git clone https://github.com/clutchlb3/abod-coffee-shop.git
cd abod-coffee-shop
npm install
npm run dev
